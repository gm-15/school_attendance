# 학교 출석 프로그램 테스트 체크리스트

## 📋 테스트 가이드

이 문서는 프로젝트의 모든 기능을 체계적으로 테스트하기 위한 체크리스트입니다.
각 항목을 테스트하고 결과를 기록하세요.

---

## 1. 인증/인가 시스템 ✅

### 로그인/로그아웃
- [ ] 관리자 계정으로 로그인 성공
- [ ] 교원 계정으로 로그인 성공
- [ ] 학생 계정으로 로그인 성공
- [ ] 잘못된 비밀번호로 로그인 시도 → 오류 메시지 표시
- [ ] 존재하지 않는 이메일로 로그인 시도 → 오류 메시지 표시
- [ ] 로그아웃 버튼 클릭 → 로그인 페이지로 이동
- [ ] 새로고침 후에도 로그인 상태 유지 (쿠키 기반)
- [ ] 다른 탭에서 로그아웃 → 다른 탭도 로그아웃 처리

### 권한 확인
- [ ] 학생이 관리자 페이지 접근 시도 → 차단 또는 리다이렉트
- [ ] 교원이 다른 교원의 과목 접근 시도 → 차단
- [ ] 학생이 다른 학생의 출석 정보 접근 시도 → 차단
